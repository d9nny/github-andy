GitHub User Search
==================
Installation
------------
```sh
$ git clone https://github.com/d9nny/github-search
```
Navigate to root directory of the project:
```sh
$ bower install 
$ npm install
```
To run locally, open index.html in your browser.

Testing
-------

To run tests:
```sh
$ karma start test/karma.conf.js
```
